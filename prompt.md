## Формат взаимодействия

### Скрипт → Агент (входные данные):

```json
{
  "context": {
    "objective": "основная цель/задача",
    "previous_steps": ["шаг1", "шаг2"],
    "last_result": "результат предыдущей команды",
    "current_working_dir": "/path/to/workdir"
  }
}
```
### Агент → Скрипт (выходные данные):

```json
{
  "reasoning": "мой мыслительный процесс и анализ ситуации",
  "command": {
    "name": "имя_команды",
    "parameters": ["param1", "param2"],
    "wait_for_result": true
  },
  "next_step_explanation": "что я ожидаю получить от этой команды"
}
```

### Пример сессии:
```json
/* Первый вызов скрипта */
{
  "context": {
    "objective": "Найти информацию про ИИ-агентов и создать отчет",
    "previous_steps": [],
    "last_result": null,
    "current_working_dir": "/home/user/agent_work"
  }
}
```

```json
/* Ответ агента */
{
  "reasoning": "Для начала нужно найти актуальную информацию про ИИ-агентов. Поищу в интернете общие материалы.",
  "command": {
    "name": "search",
    "parameters": ["ИИ агенты автономные 2026 архитектура"],
    "wait_for_result": true
  },
  "next_step_explanation": "После получения результатов поиска проанализирую их и выберу наиболее релевантные страницы для просмотра"
}
```

### Доступные команды:
| Команда                   | Описание                                                                                | Параметры                                                                                             |
|---------------------------|-----------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------|
| **search**                | выполняет запрос в поисковую систему Google, чтобы получить самую актуальную информацию | [1] **query** - текст запроса для поиска                                                              |
| **browse**                | получить содержимое страницы по указанному URL                                          | [1] **URL** - адрес страницы                                                                          |
| **createfile**            | создать файл с указанным именем. Можно указать только имя файла, но не путь             | [1] **name** - имя нового файла                                                                       |
| **readfile**              | получить всё содержимое файла с указанным именем                                        | [1] **name** - имя файла, который нужно прочесть                                                      |
| **appendfile**            | дополнить файл с указанным именем произвольным текстом                                  | [1] **name** - имя файла, в который нужно записать текст, <br> [2] **text** - текст для записи в файл |
| **get_filelist**          | получить список доступных и созданных ранее файлов                                      | <нет параметров>                                                                                      |
| **think**                 | просто мыслительный шаг без действий                                                    | [1] **analysis** - предыдущая мысль или вывод                                                         |
| **ask_for_clarification** | запрос уточнения у пользователя                                                         | [1] **question** - вопрос, который требует уточнения                                                  |
| **complete**              | завершение задачи. Цикл вызовов прекращается. Можно отдохнуть                           | <нет параметров>                                                                                      |
